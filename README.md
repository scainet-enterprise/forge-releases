# SCAINET Forge — Release Binaries

Public distribution mirror for [SCAINET Forge](https://scainet.io) release assets.

This repository contains only compiled binaries and update manifests.  
Source code is maintained in a separate private repository.

## Downloads

See [Releases](https://github.com/scainet-enterprise/forge-releases/releases) for the latest installers.

| Platform | Format |
|----------|--------|
| Windows (x64) | `.exe` (NSIS installer) |
| macOS (ARM64) | `.dmg` |
| Linux (x64) | `.deb`, `.AppImage` |
